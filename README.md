# WVU ETD LaTeX Template (multi-file)

This template implements WVU's key formatting rules:

- Title page = page i, but page number suppressed
- Abstract page(s) = page ii (and iii if it runs long), but page numbers suppressed
- Front matter after abstract uses lower-case Roman numerals (iii, iv, v, ...)
- Main body restarts numbering at Arabic 1

Replace bracketed placeholders (remove the brackets).
